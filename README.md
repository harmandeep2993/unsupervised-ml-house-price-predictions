# ml-house-price-analysis & predictions

## Project Objective
The objective of this project is to analyze house prices using unsupervised learning techniques and identify patterns and clusters in the data.

## Project Scope
This project includes data cleaning, exploratory data analysis, clustering, and PCA. Supervised learning models are not included.

## Project Steps
1. Data understanding
2. Data cleaning and preparation
3. Exploratory data analysis
4. K-Means clustering
5. PCA visualization
6. High-value house analysis
7. Results and insights

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
